# Foblex Flow — Minimal Starter

A minimal Angular app with a working node editor: two nodes, drag to move, drag between connectors to create a connection.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Foblex/f-flow/tree/main/starters/minimal-flow)

```bash
npm install
npm start
```

Docs: https://flow.foblex.com/docs/get-started
